# WDCCxSESA Hackthon Website!

This repository hosts the in progress codebase for WDCCxSESA Hackathon website! <br/>

## Getting Started

```bash
# cloning the project
git clone https://github.com/sesauoa/wdccxsesahackathon.com.git.

# change directory
cd .\wdccxsesahackathon.com\

# install dependencies
pnpm install

# run development server
pnpm run dev
```
